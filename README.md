# 🚗 Vehicle Booking Application

A full-stack Vehicle Booking Application that enables users to register, log in, browse available vehicles, book vehicles for specific dates, and manage bookings. Administrators can manage vehicles and view all bookings through a secure dashboard.

---

## Features

### User
- User Registration and Login
- JWT Authentication
- Browse Available Vehicles
- Book Vehicles
- View My Bookings
- Cancel Bookings

### Admin
- Add New Vehicles
- Update Vehicle Details
- Delete Vehicles
- View All Bookings
- Manage Vehicle Availability

## Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/Vehicle-Booking-Application.git
```

### Backend

```bash
cd VehicleBookingAPI
```

- Update the SQL Server connection string in `appsettings.json`
- Run the database scripts
- Start the ASP.NET Core Web API

### Frontend

```bash
cd VehicleBookingClient
npm install
npm run dev
```
